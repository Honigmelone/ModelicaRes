These Python modules can be used to analyze and plot results from Modelica
simulations.  More information is available in the HTML or PDF documentation
(help/index.html or help/modelicares.pdf), which is also available online at
http://kdavies4.github.com/modelicares.

To install on Linux, do the following from the base folder:
1. Build the modules.
   $ python ./setup.py build
2. Install the modules.
   $ sudo python ./setup.py install
